0. User Registration and Profile creation:
GOAL:
Establish user authentication and profile management functionality within the web application to provide users with a secure and personalized experience.

  1. 1. Create Registration page:
    1. 1.1 Create input fields.
      1. 1.1.1 Create input for first name.
      2. 1.1.2 Create input for last name.
      3. 1.1.3 Create input for email.
      4. 1.1.4 Create input for password.
      5. 1.1.5 Create input for user id.
    2. 1.2 Create validation for user id availability.
    3. 1.3 Create validation for password.
    4. 1.4 Create the “Sign Up" button.
  2. 2. Create login page:
    1. 2.1 Create input for user id.
    2. 2.2 Create input for password.
    3. 2.3 Create validation.
    4. 2.4 Create the "Login" button.
  3. 3. Create User Profile:
    1. 3.1 Create a link between login and user profile page.
    2. 3.2 Create a display for user details.
    3. 3.3 Create "Edit Profile." Button.
    4. 3.4 Create “Edit Profile” page.
      1. 3.4.1 Create input for age.
      2. 3.4.2 Create input for preferences.
      3. 3.4.3 Create input for location.
      4. 3.4.4 Create input for profile photo.
      5. 3.4.5 Create “save changes” button.
    5. 3.5 Create a link between the edit profile button and “Edit profile” page.
    6. 3.6 Update edited content on User profile page.
    7. 3.7 create a display for Volunteer History.
    4. Log Out:
      4.1 Create logout button. 4.2 Link logout button to home page

  1. 1. Create a page for Volunteer Posting:

    2. 1.2 Build “Create opportunity” page.

      2. 1.2.2 Create an input for Location.
      3. 1.2.3 Create an input for Time.
      4. 1.2.4 Create an input for Skills Required.
      5. 1.2.5 Create an input for contact email.
      6. 1.2.6 Create an input for the number of volunteers needed.
      7. 1.2.7 Create an input for perks.
      8. 1.2.8 Create the “Post” button.
    3. 1.3 Create a link between the Create Opportunity button and “Create Opportunity” Page.
    4. 1.4 Create a link between the Post button and the volunteer posting page.
    5. 1.5 Upload the content on volunteer posting page.
  2. 2. Edit a Volunteer Post:
    1. 2.1 Create a button “Edit Opportunity”.
    2. 2.2 Build the “Edit Opportunity” page.
      1. 2.2.1 Create an input for Date.
      2. 2.2.2 Create an input for Location.
      3. 2.2.3 Create an input for Time.
      4. 2.2.4 Create an input for Skills Required.
      5. 2.2.5 Create an input for contact email.
      6. 2.2.6 Create an input for the number of volunteers needed.
      7. 2.2.7 Create an input for perks.
      8. 2.2.8 Create the “Post” button.
    3. 2.3 Create link between Edit Opportunity button and Edit Opportunity Page
    4. 2.4 Create a link between the Post button and the volunteer posting page.

    6. 2.6 Update the respective volunteers.
  3. 3. Delete a Volunteer Post:
    1. 3.1 Create a button “Delete Opportunity”.

0. Events.
GOAL:
Implement a comprehensive and user-friendly search feature within the web application to enable users to explore and discover volunteer opportunities that match their preferences and needs effectively.
  1. 1. Implement the Search feature:
    1. 1.1 Create search bar.
    2. 1.2 Add a search icon or button next to the search bar for initiating the search.
    3. 1.3 Write a search algorithm.
  2. 2. Results Display Area:
    1. 2.1 Design a block for displaying search results.
    2. 2.2 Organize results (volunteer opportunities) in a grid/card.
    3. 2.3 Make the result cards actionable to trigger the opportunity details page.
  3. 3. Filters:
    1. 3.1 Design the filters using sliders/checkboxes.
    2. 3.2 Update the display according to the results.
    3. 3.3 Display error messages.
Plans:
  1. Plan 0: Do 1,2, 3 in any order.
  2. Plan 1: Do 1.1 to 1.3 in the same order.
  3. Plan 2: Do 2.1 to 2.3 in the same order.
  4. Plan 3: Do 3.1 to 3.3 in the same order.
